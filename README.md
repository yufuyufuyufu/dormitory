# 学生宿舍管理系统
**本项目旨在管理学生公寓数据，实现宿舍管理、学生管理、宿舍分配、宿舍评比、损坏管理查询等必需的功能**  
> + 现如今大学校园内的学生公寓数量多，各类信息管理内容复杂，查询和管理学生信息和校园内的公寓信息等人工工作量较大，浪费了许多人力物力资源，且容易出现差错；
> + 因此我们计划开发一款软件对学生公寓数据进行管理，便于对各种信息的提取和操作。
